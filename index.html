Toets JavaScript periode 3

Welkom bij de JavaScript toets. Er zijn 5 opdrachten, voor elke opdracht lees je goed wat de bedoeling is op die pagina.
Als je een variabele maakt kies jezelf voor let, var of const. Let op dat je de juiste keuze maakt die past bij de opdracht/ code.
Je mag het internet gebruiken bij deze toets.
Je kan 80 punten voor deze toets halen. 55 voor de opdrachten en 25 voor de netheid/ correctheid van jouw code.
Jouw code wordt ook beoordeeld op de volgende punten:
  • Netheid van de code - 5 punten
  • Correct indenten van de code - 5 punten
  • Correct gebruik en positie van { } [ ] tekens in jouw code
  • Gebruik van camelCase bij functies en variabelen benamingen - 5 punten
  • Engelse taal van functies en variabelen benamingen - 5 punten
